项目背景

这是 2026 年度 CCF 量子计算编程挑战赛（本源杯）相关项目经历。项目主要围绕量子机器学习赛题展开，尤其是第三题的 VQNet 训练、测试入口和实验记录。

我的工作

  • 参与第三题 VQNet 训练、测试入口和实验记录链路的工程化工作。
  • 协助维护模型 artifact、experiment log、summary 工具和 GPU / WSL 运行说明。
  • 参与量子线路变体、训练参数和 validation 结果的整理与复盘。

技术方法

项目使用 Python 3.10、pyqpanda3、pyvqnet 和 VQNet GPU 训练链路。工程上重点是把训练入口、测试入口、模型 artifact 和实验记录收束成可复查流程，而不是只保留零散 notebook 或一次性脚本。

结果与边界

已确认获得 2026 年度 CCF 量子计算编程挑战赛（本源杯）三等奖。
