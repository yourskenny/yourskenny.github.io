## 项目简介

该项目将课程资料整理为可检索知识库，支持基于资料来源的问答。你可以把它替换成自己做过的 LLM、知识库或智能助手项目。

## 技术栈

- Python
- 向量数据库或本地向量索引
- Embedding 模型
- Prompt 设计与问答评估

## 我的工作

我负责文档清洗、文本切分策略、检索链路设计和回答质量评估。

## 方法与实现

项目可以按“文档导入 -> 切分 -> 向量化 -> 检索 -> 生成 -> 引用来源”的流程说明。重点写清楚你如何评估回答是否可靠。

## 结果展示

建议补充 3 到 5 个真实问答样例，包括回答正确、部分正确和失败的案例。

## 难点与解决

RAG 项目常见难点是检索到的片段不够相关。可以记录你尝试过的 chunk size、top-k、rerank 或关键词混合检索策略。

## 复盘
