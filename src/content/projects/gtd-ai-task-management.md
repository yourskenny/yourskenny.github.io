## 项目简介

GTD Pro 是一个基于 Getting Things Done 方法论的现代化时间管理系统，核心流程是随时捕获、智能整理和聚焦执行。

## 技术栈

- Python / FastAPI
- React / TypeScript / Vite
- SQLite
- OpenAI / DeepSeek API
- PowerShell 一键开发启动脚本

## 我的工作

我搭建了前后端分离的开发结构，并将 LLM 引入任务整理流程，用于把自然语言输入转换为更结构化、可执行的任务管理信息。

## 方法与实现

后端提供 API 和数据管理能力，前端提供任务捕获与组织界面。项目包含一键启动脚本，自动创建 Python 环境、安装前后端依赖并启动服务。

## 难点与解决

GTD 类应用的难点在于降低记录摩擦，同时保证信息足够结构化。AI 的价值不在于替代用户规划，而在于把碎片输入快速整理为可执行对象。
